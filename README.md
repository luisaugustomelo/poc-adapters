### POC Adapters

## Install


- using yarn
```
    yarn
```

- using npm
```
    npm install
```

## Running

- using yarn
```
    yarn start
```

- using npm
```
    npm start
```


## Structure

```
    dist/
    src/
        adapters/
            RandomLogAdapter.ts
            TodoAdapter.ts
        interfaces
            IAdapter.ts
        adapter.ts
        main.ts
```
